# WLJS Notebook Plugin Template

A boilerplate for writting various plugins/extensions for WLJS Notebook

Clone it to `<AppData>/wljs_packages/` folder and restart an app

## Commands

To bundle Javascript

```bash
npm i
npm run build
```

To restore `.git` 

```bash
npm run restore
```
